# Roadmap

## Next public milestone

- Publish the project in a dedicated GitHub repository and deploy the deterministic demo.
- Add a Salesforce Developer Edition integration using least-privilege OAuth and synthetic records.
- Replace the in-memory approval store with durable persistence and enterprise identity.
- Add framework-native LangGraph integration tests with a persistent checkpointer.
- Add OpenTelemetry traces and an MLflow experiment dashboard.

## Later architecture demonstrations

- Hybrid BM25 and vector retrieval with measurable relevance comparison.
- Credentialed RAGAS and DeepEval runs as non-blocking supplemental evidence.
- PySpark/Databricks ingestion validation on a synthetic large-data workload.
- Cloud-specific deployment examples for Azure, AWS, and Google Cloud.
- Load, reliability, model-cost, and failure-injection testing.

Roadmap items are not represented as completed capabilities until code, tests, and reproducible evidence are checked in.
