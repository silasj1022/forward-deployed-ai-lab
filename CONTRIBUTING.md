# Contributing

1. Do not add proprietary, employer, customer, government, or student data.
2. Keep the default workflow reproducible without paid model credentials.
3. Label integrations honestly as implemented, optional, or planned.
4. Add tests and update the golden set for behavior changes.
5. Run `make lint test evaluate red-team` before opening a pull request.
6. Keep consequential actions approval-gated and auditable.
